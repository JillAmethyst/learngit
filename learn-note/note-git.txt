创建分支:
  1: git checkout -b dev 创建一个名为dev的branch
     (
       git checkout命令加上-b参数表示创建并切换，相当于以下两条命令:
       $ git branch dev
       $ git checkout dev
      )
  2: git branch 查看当前有哪些分支
     git checkout dev 可以切换到branchname的那个分支上
     (在不同的分支上用ls命令可以检验不同分支上内容不同)

  3: git merge dev 合并dev分支到master分支上
     (合并后dev的内容就会和master的内容融合)
